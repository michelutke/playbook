## How to check CI
```bash
source ~/.openclaw/.env
curl -s -H "Authorization: Bearer $GITHUB_TOKEN" \
  "https://api.github.com/repos/michelutke/playbook/actions/runs?per_page=3&branch=feat/phase-2-team-management" \
  | python3 -c "
import sys, json
data = json.load(sys.stdin)
for r in data.get('workflow_runs', []):
    print(f\"{r['conclusion'] or r['status']:12} | {r['name'][:30]} | {r['id']}\")
"
```

## How to get job errors
```bash
source ~/.openclaw/.env
RUN_ID=<id>
curl -s -H "Authorization: Bearer $GITHUB_TOKEN" \
  "https://api.github.com/repos/michelutke/playbook/actions/runs/$RUN_ID/jobs" \
  | python3 -c "import sys,json; [print(j['id'], j['name']) for j in json.load(sys.stdin).get('jobs',[])]"

# then for each job:
curl -sL -H "Authorization: Bearer $GITHUB_TOKEN" \
  "https://api.github.com/repos/michelutke/playbook/actions/jobs/$JOB_ID/logs" | \
  grep -v "##\[group\]\|##\[endgroup\]\|Download\|cache" | \
  grep -E "e: file://|Unresolved|FAILED|error:" | head -20
```
