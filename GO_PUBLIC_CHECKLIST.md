## 1. Secret Scan — Git History

203 commits exist. Even if no secrets are in the current tree, they may be in history.
A secret in any commit = permanent exposure unless history is rewritten.

```bash
# Install trufflehog (fastest, understands git history)
brew install trufflehog

# Scan full history
trufflehog git file://. --since-commit HEAD~203 --only-verified
```

If trufflehog flags anything:
- Rotate the credential immediately (before doing anything else)
- Use `git filter-repo` to scrub the file/string from history (requires force-push)
- All collaborators must re-clone after history rewrite

Alternatively use `git-secrets` or `gitleaks` as a second opinion:
```bash
brew install gitleaks
gitleaks detect --source . --log-opts="HEAD~203..HEAD"
```

**Expected clean:** passwords/keys are read from `System.getenv()` and GitHub Secrets,
not hardcoded. Verify the scan confirms this.

## 2. Current-Tree Secret Scan

```bash
# Grep for common patterns across all source files
grep -rn --include="*.kt" --include="*.swift" --include="*.gradle.kts" \
  --include="*.yml" --include="*.yaml" --include="*.json" \
  -iE "(password|secret|token|api_key|apikey|private_key|bearer)\s*[=:]\s*['\"][^'\"]{6,}" \
  . | grep -v "build/" | grep -v "System.getenv" | grep -v "secrets\." | grep -v "vars\."
```

**Known safe (already verified):**
- `build.gradle.kts` keystore passwords → `System.getenv("ANDROID_KEYSTORE_PASSWORD")` ✓
- CI workflows → all sensitive values via `${{ secrets.* }}` or `${{ vars.* }}` ✓
- No hardcoded API base URLs in source (injected via `local.properties` at build time) ✓

## 5. Add a LICENSE File

Required for open source. Without one, the code is technically "all rights reserved"
even if the repo is public.

Choose one:
- **MIT** — permissive, simplest, most common for apps
- **Apache 2.0** — permissive + explicit patent grant (better for libraries)
- **AGPL-3.0** — copyleft, requires derivatives to also be open source

```bash
# Example: MIT
cat > LICENSE << 'EOF'
MIT License

Copyright (c) 2026 <your name / org>

Permission is hereby granted, free of charge, to any person obtaining a copy
of this software and associated documentation files (the "Software"), to deal
in the Software without restriction, including without limitation the rights
to use, copy, modify, merge, publish, distribute, sublicense, and/or sell
copies of the Software, and to permit persons to whom the Software is
furnished to do so, subject to the following conditions:

The above copyright notice and this permission notice shall be included in all
copies or substantial portions of the Software.

THE SOFTWARE IS PROVIDED "AS IS", WITHOUT WARRANTY OF ANY KIND, EXPRESS OR

EOF
git add LICENSE && git commit -m "chore: add MIT license"
```
