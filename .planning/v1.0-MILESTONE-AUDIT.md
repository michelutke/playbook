## Cross-Phase Integration Issues

### P0 — Runtime Failures

**BREAK-01: CheckInEntry shape mismatch (AT-04, AT-11, AT-12)**
Server `GET /events/{id}/check-in` returns `List<CheckInRow>` (flat fields). Client deserializes as `List<CheckInEntry>` (nested objects + userName/userAvatar). Ktor deserialization will fail → member response list always empty → coach override inaccessible.

**PARTIAL-01: TeamRolesTable Exposed object not nullable (TM-19, AT-13)**
V8 migration made `team_roles.user_id` nullable + SET NULL. But `TeamRolesTable.kt` still declares it non-nullable + CASCADE. Queries reading NULL user_id after member removal → NPE.

### P1 — Degraded Features

**PARTIAL-02: Empty eventTypes map (TM-17, AT-05)**
`PlayerProfileViewModel.loadStats` passes `emptyMap()` to `calculateStats`. Per-type breakdown (training/match) always returns 0%.

**PARTIAL-03: Profile navigation gap (TM-17, AT-05, AT-06, AT-08)**
Bottom nav Profile tab → `PlaceholderScreen`. PlayerProfileScreen (with attendance features) only reachable via Teams → Roster → member tap.
